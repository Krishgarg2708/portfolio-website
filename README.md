#  Personal Portfolio Website  

A simple and responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript**.  
This project showcases your personal information, skills, and projects — perfect for students and beginners learning web development.  

---

##  Features
- Clean and modern design  
- Fully responsive layout  
- Separate sections for About, Skills, Projects, and Contact  
- Easy to customize (just edit the HTML & CSS)  
- Hosted using **GitHub Pages**  

---

##  Technologies Used
- **HTML5** – for structure  
- **CSS3** – for styling and layout  
- **JavaScript** – for interactivity
